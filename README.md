## About the Project

**Snake Road: Safe Traffic Journey** is a Python game built using **Pygame**.
In this game, a boy travels from **Home to School** on a snake-shaped road while avoiding vehicles and making decisions about stopping at different locations.

The project demonstrates:

* Game development using Python
* Event handling and collision detection
* Object-oriented programming
* Real-time traffic simulation
* Interactive UI design

---

## Features

* Snake-shaped dynamic road
* Moving traffic (cars, bikes, autos)
* Collision detection system
* Optional stops:

  * Market
  * Mall
  * Friend‚Äôs House
* Menu system and game states
* Return home option after reaching school
* Restart after accident

## Controls

| Key         | Action                 |
| ----------- | ---------------------- |
| Right Arrow | Move forward           |
| Left Arrow  | Move backward          |
| 1           | Direct to school       |
| 2           | Stop at locations      |
| Space       | Continue after stop    |
| Y           | Return home            |
| R           | Restart after accident |
| ESC         | Exit game              |

## Installation

1. Clone the repository:

```

```

3. Run the game:

